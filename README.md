## About
This is the frontend application for the ECA project, built using Next.js. It serves as the user interface for managing the e-commerce and POS system.

## Description
The ECA frontend is a modern web application that provides a comprehensive dashboard for managing products, customers, orders, and point-of-sale (POS) operations. It is designed to offer a seamless experience for administrators.

## Features
- **Dashboard Overview:** Centralized view of system metrics and quick actions.
- **Point of Sale (POS):** Streamlined interface for processing walk-in customer interactions.
- **Product Management:** Create, read, update, and delete product listings.
- **Order Management:** Manage and track customer orders.
- **Customer Management:** View and organize customer information.
- **Secure Authentication:** Admin login and registration workflows.
- **Responsive Design:** Optimized for both desktop and mobile viewing.

## Setup Guides

### Prerequisites
- Node.js 18.x or later
- npm, yarn, pnpm, or bun

### Installation

1. Navigate into the project directory:
   ```bash
   cd eca-frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

### Running the Development Server

Start the Next.js development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
